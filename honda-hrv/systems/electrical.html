Body Electrical

Two fuse/relay boxes: engine compartment (left side) and under-dash (left of steering column). Body electrical controlled by integrated control unit and ECM/PCM.

Fuse & Relay Boxes

Under-hood fuse box: Left side, near battery. Contains high-current fuses for engine, cooling fan, ABS, and main relays.
Under-dash fuse box: Left of steering column, behind kick panel. Contains body electrics fuses and relays.

Power Windows

Electric lift with motor in each door. Driver's switch has master control. Auto-down function on driver's window (KG/KE models).

Window Motor Test

Disconnect motor connector. Apply 12V to terminals (either polarity for direction). If motor doesn't run, replace. If motor runs but window doesn't move: regulator cable broken or guide seized.

Common Faults

Window won't move: Check fuse first. Test motor direct. If motor OK: switch fault or wiring break in door harness (check rubber boot between door and A-pillar).
Window slow/struggles: Dry guide rails (lubricate with silicone spray), weak motor, poor earth connection.

Power Door Locks

Actuator in each door controlled by driver's switch or keyless remote (where fitted). Tailgate lock operates via separate actuator.

Test: Listen for actuator click when locking. If one door doesn't lock: test actuator with 12V direct. If works: wiring or switch fault. If doesn't: actuator failed (common).

SRS (Supplemental Restraint System)

Dual front airbags (driver and passenger on some models). SRS unit under centre console. Yellow wiring harnesses - do not test with multimeter on resistance (can trigger deployment).

SRS Safety Warning

Never use a multimeter on resistance mode to test SRS circuits - this can deploy the airbag. Only use a dedicated SRS tester or Honda PGM tester. Disconnect battery and wait 3 minutes before working on SRS components. Store airbag module with trim cover facing up.

Immobiliser System

Honda transponder immobiliser standard on most models. Prevents engine starting without registered key.

ComponentLocation
Immobiliser receiverAround ignition barrel
Immobiliser indicatorDashboard (red key symbol)
ECU controlIntegrated with PGM-FI ECM

Symptom: Engine cranks but won't start, immobiliser light flashing. Cause: Unregistered key, faulty transponder in key, receiver fault, or ECM mismatch. Fix: Try spare key. If same: requires Honda diagnostic tool to recode or replace receiver and re-register keys.

Relay Notes - HVAC Blower

The HR-V HVAC blower motor is controlled via a power transistor (not a simple relay). Common failure: transistor overheats, causing blower to work only on high speed or not at all. If blower works on max only: power transistor has failed. Located behind glovebox.

Rear defogger relay: Similar to VW Golf, the rear window defogger can benefit from a relay upgrade if the factory switch gets hot. Add a 30A relay triggered by the existing switch, with a new 4.0 mm² feed direct from battery via 25A fuse.

Body Electrical Quick Diagnosis

Multiple circuits dead: Check under-dash fuse box for corroded fuse clips (common in humid climates).
Intermittent electrics: Check door harness rubber boots for broken wires.
Battery drains overnight: Check parasitic drain (should be under 50mA). Pull fuses one by one to find circuit.
Immobiliser light flashing, no start: Try spare key. If same: needs Honda diagnostic.
