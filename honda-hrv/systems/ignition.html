Engine Electrical

12V negative earth. Distributor ignition with external ICM (Ignition Control Module) on D16W1, integrated ICM on D16W5. All engines use OBD-II diagnostics.

Starter Solenoid Test

1. Disconnect ignition coil to prevent starting. 2. Connect voltmeter between solenoid S terminal and earth. 3. Crank engine. Should read battery voltage (12V+). 4. If voltage OK but starter doesn't crank: solenoid or motor faulty.

Brush limit: Minimum length 7.0 mm. Armature runout: Max 0.05 mm.

Ignition Coil

Location: On cylinder head, near distributor.
Primary resistance: 0.45-0.55 ohms
Secondary resistance: 16.5-25.0 kΩ
Test: Remove all wires. Measure primary across + and - terminals. Measure secondary between + terminal and tower. Replace if out of spec.

Ignition Control Module (ICM) - D16W1

Location: Mounted on distributor body (external).
Function: Controls ignition coil firing based on CKP sensor signal.
Input test: With ignition ON: White/Black wire = 12V. Black/Yellow = 12V. White/Blue = signal to coil. Green/White = CKP signal (pulsing while cranking).
Replacement: Apply heat-sink compound to mating surface.

CKP/TDC/CYP Sensor (D16W5)

Combined crankshaft position, top dead centre, and cylinder position sensor in distributor. Creates ignition timing signal and cylinder identification.

Test: Measure resistance across sensor terminals. Should be 350-700 ohms. Check for signal output while cranking (AC voltage or oscilloscope).

Alternator Test

1

Check drivebelt tension and condition.

2

Measure battery voltage with engine off: 12.4V+ (fully charged).

3

Start engine, run at 2000 rpm. Measure battery voltage: should be 13.5-15.1V.

4

Turn on headlights, heater fan, rear defogger. Voltage should remain above 13.0V. If not, alternator or regulator failing.

Testing Procedures

Ignition Timing Check

1

Connect SCS service connector (brown 2-pin under dash, right side) - use paperclip to jump pins.

2

Start engine. Connect timing light to No.1 cylinder.

3

Check timing at idle: 8-12° BTDC (red mark on crank pulley). If incorrect, check base timing and distributor installation.

4

Disconnect SCS jumper. Timing should advance to 16-24° BTDC (blue mark). If no advance, check ECM and sensor inputs.

Engine Electrical Quick Diagnosis

Crank but no start: Check for spark (remove plug, ground to engine, crank). No spark: check ICM, CKP sensor, coil. Has spark: check fuel pressure, injector operation.
Hard start cold: Check coolant temp sensor (ECT), ignition timing.
Engine stalls hot: Check ICM (thermal failure), main relay (cracked solder), fuel pump relay.
Battery light on: Check alternator output voltage, drivebelt, battery terminals.
