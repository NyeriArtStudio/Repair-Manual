Engine Mechanical

All Honda HR-V engines are 1.6L SOHC 16-valve inline-4 with Honda PGM-FI fuel injection. Water-cooled, transverse mount, aluminium alloy cylinder head and block.

CRITICAL: Interference Engine

Timing belt failure WILL cause valves to hit pistons. Expensive engine damage is guaranteed. Do not exceed the replacement interval. If the engine has been sitting unused for 4+ years, replace the belt regardless of mileage (rubber degrades with age).

Timing Belt Installation

1

Set engine to TDC: align crankshaft pulley TDC mark with pointer. Camshaft pulley "UP" mark should be at top.

2

Remove crankshaft pulley (245 Nm bolt - special holder tool required). Remove upper and lower timing covers.

3

Loosen tensioner, remove old belt. Check all idler pulleys for smooth rotation and noise. Renew if any roughness.

4

Install new belt starting at crankshaft, then camshaft pulley. Keep slack on tensioner side.

5

Tension belt according to automatic tensioner instructions (or manual tensioner: apply specified torque). Rotate engine two full turns. Recheck TDC alignment and belt tension.

6

Install covers, crankshaft pulley (245 Nm). Refill coolant. Run and check for leaks.

VTEC System (D16W5)

VTEC (Variable Valve Timing and Lift Electronic Control) switches between two cam profiles at high rpm. Below 4800 rpm: standard profile for economy. Above 4800 rpm: high-lift/long-duration profile for power.

VTEC Solenoid Valve

Location: Rear of cylinder head, single-wire connector.
Function: Controls oil pressure to VTEC rocker arm mechanism.
Test: Remove valve. Apply 12V to terminal, ground body. Should hear distinct click. Check oil filter screen for debris. Clean with brake cleaner if clogged.
OEM: 15810-P2E-A01
Screen gasket: 15815-P08-005

VTEC Rocker Arms

Each cylinder has three rocker arms: two standard (outer) and one high-lift (centre). At VTEC engagement, a synchronising pin locks all three together so the high-lift cam operates the valves.

Manual inspection: With engine cold and off, remove valve cover. Press on each rocker arm - standard rockers should have slight free play. VTEC rocker should move independently below engagement. At 3000+ rpm (running test), all three should lock together.
Pressure test: Remove VTEC solenoid and apply 50 psi oil pressure to the port. Rocker pins should engage smoothly.

Testing Procedures

Compression Test

1

Engine at operating temperature. Remove all spark plugs. Disable fuel injection (remove PGM-FI main relay fuse).

2

Install compression gauge. Throttle fully open. Crank 5 revolutions.

3

Minimum pressure: 930 kPa (135 psi). Max variation between cylinders: 200 kPa (29 psi). Standard: 1200+ kPa.

Oil Pressure Test

Remove oil pressure sender from oil filter housing. Install gauge. At 3000 rpm, oil hot: minimum 70 kPa (10 psi). *Standard value - check against Honda spec for your model

Cylinder Head Bolts

Tighten in sequence: 29 Nm, then 64 Nm, then 90° turn. Bolts are torque-to-yield (single use). Always replace with new Honda bolts. Failure to do so risks head gasket failure.
